# Uptime

An realtime server status monitor that interfaces with Nodeping.

## Prerequisite

  - Node.js
  - Sails.js

## Install

### Install dependencies

		npm install

### Setup configuration

		cp config/default.yml.bak config/default.yml

## Start

		sails lift

## Deployment

  See sailsjs official doc for details.
